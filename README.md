## What WineBOX is

WineBOX is a lightweight digital space for Android that favors readability and predictability instead of a busy visual design. Its low-fidelity desktop model is meant to cut down on distraction while still leaving room for interactive use, cognitive stimulation, and executive function practice.

The project is especially relevant for accessibility-centered scenarios, including neurodiversity and autism support, where stable interactions and less sensory input can be useful. A WebAssembly-powered web demo is also included, so the concept can be explored in a browser before moving to the Android build.
